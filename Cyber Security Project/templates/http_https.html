💡 Why is this dangerous?

When using HTTP, data is sent as plain text. Any attacker on the same network (public Wi-Fi, compromised router) can use tools like Wireshark to intercept and read your credentials instantly.

Impact: Credential Theft, Session Hijacking.

🛡️ SOC Recommendations

  • Vulnerability: Use of unencrypted communication channel (Port 80).
  • Risk: HIGH. Complete loss of data confidentiality.
  • Mitigation: Enforce HSTS and ensure all traffic is routed via HTTPS (TLS 1.2/1.3).
